Research Methodology

  • Integrated environmental sensors with an Arduino-based Data Acquisition System (DAS).
  • Implemented telemetry modules for wireless communication between UAV and ground station.
  • Mounted the DAS on a quadcopter and conducted multiple test flights.
  • Live sensor data was transmitted to the ground station in real-time via telemetry link.
  • Captured data was processed and visualized into 2D/3D graphs using Python scripts.

Conclusion

  • Successfully captured live environmental data such as temperature and humidity during flight.
  • Enabled real-time decision-support capabilities through live telemetry and data visualization.
  • Demonstrated potential for future UAV-based missions like pollution source detection and wildlife monitoring.
  • The integration of DAS into UAV platforms is a key enabler for autonomous, data-driven decision-making.
