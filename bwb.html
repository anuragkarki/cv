Optimization of BWB UAV

Research Methodology

    • Low-fidelity aerodynamic analysis was performed using XFLR5 and   OpenVSP.
    • Initially, gradient optimization was performed using SU2.
    • Then non-gradient optimization was performed using Aeolus.
    • High-fidelity aerodynamic analysis was done for baseline and optimized model using ANSYS ICEM and fluent.
    • Comparison was done between baseline model and optimized model.

Research Results

    • Gradient-based optimization resulted in 6% increment in lift/drag ratio
    • Non-gradient optimization resulted in 23% increment in lift/drag ratio.
    • Finally, some manual modifications were made based on literature,usability and expert feedback which resulted in a 3% increment in lift/drag ratio
